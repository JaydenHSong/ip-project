🏗️ 3개 컴포넌트 스펙
🔍
Sentinel Crawler
  • Playwright 브라우저 자동화
  • Anti-bot 회피 (프록시, Fingerprint, 랜덤 행동)
  • 키워드 캠페인 기반 1~5페이지 수집
  • 다국가 지원 (US, UK, JP 등)
  • 의심 리스팅 키워드 플래그 자동 필터링
  • 신고 후 리스팅 자동 재방문 (팔로업)
  • 스케줄링: 사용자 설정 빈도
🧩
Sentinel Extension
  • Chrome Extension (.crx 사내 배포)
  • 아마존 상품 페이지 플로팅 버튼
  • 원클릭 정보 자동 캡처 (DOM 파싱)
  • 위반 유형 선택 + 메모 입력 UI
  • 스크린샷 자동 캡처 (증거 보존)
  • Sentinel Web API 연동
  • 대상: 오퍼레이터 20명+
🌐
Sentinel Web
  • Google OAuth (@spigen.com 한정)
  • RBAC 3단계: Admin / Editor / Viewer
  • 신고 대기열 통합 관리
  • AI 분석 결과 + 신고서 드래프트 표시
  • 검토/승인 워크플로우
  • 대시보드/리포트/트렌드
  • Changelog (버전 관리 로그)
