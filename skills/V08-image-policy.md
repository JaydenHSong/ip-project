# V08 — Image Policy Violation (이미지 정책 위반)

## Detection Criteria
- Main image not on pure white background
- Text, logos, or watermarks on main product image
- Lifestyle images as main image
- Image quality below Amazon standards

## Draft Guidelines
- Specify which image policy is violated
- Reference Amazon's Product Image Requirements
- Include specific image position (main, alt 1-8)
- Request correction to comply with standards

## Evidence Requirements
- Screenshot of the violating image(s)
- Reference to specific Amazon image requirement violated

## Common Patterns
- Badges or promotional text overlay on main image
- Non-white background on main product shot
- Brand logos or watermarks on images
