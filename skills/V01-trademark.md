# V01 — Trademark Infringement (상표권 침해)

## Detection Criteria
- Unauthorized use of Spigen registered trademarks in listing title, description, or bullet points
- Misspellings or variations designed to evade trademark detection (e.g., "Spigan", "Spegen")
- Use of Spigen brand name to imply official affiliation or authorization

## Draft Guidelines
- Reference specific trademark registration numbers when available
- Quote exact text from listing that infringes the trademark
- Cite Amazon Brand Registry and IP violation policies
- Include screenshot evidence of trademark usage in context

## Evidence Requirements
- Screenshot of the listing showing trademark usage
- Trademark registration certificate reference
- Brand Registry enrollment status

## Common Patterns
- Third-party sellers using "for Spigen" or "compatible with Spigen" inappropriately
- Listings claiming to be official Spigen products without authorization
- Keyword stuffing with Spigen brand terms in backend search terms
