# V19 — Import Regulation Violation (수입 규정 위반)

## Detection Criteria
- Products imported without proper customs documentation
- Items violating trade restrictions or embargoes
- Products missing country of origin labeling

## Draft Guidelines
- Identify specific import regulation violated
- Reference CBP or relevant authority requirements
- Note country of origin discrepancies
- Cite Amazon's import compliance requirements

## Evidence Requirements
- Product listing information
- Import regulation reference
- Country of origin labeling evidence

## Common Patterns
- Missing "Made in" country labels
- Products from restricted origin countries
- Goods lacking proper import documentation
