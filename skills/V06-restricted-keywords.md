# V06 — Restricted Keyword Usage (금지 키워드 사용)

## Detection Criteria
- Usage of Amazon-restricted keywords in listing content
- Prohibited health/safety claims
- Restricted superlative terms without substantiation

## Draft Guidelines
- List each restricted keyword found with its location in the listing
- Reference Amazon's Restricted Products policies
- Cite specific keyword restriction guidelines
- Request removal of restricted terms

## Evidence Requirements
- Screenshots highlighting restricted keywords
- Amazon's restricted keyword list reference
- Location of keywords (title, bullets, description)

## Common Patterns
- Health claims ("antibacterial", "antimicrobial") without FDA approval
- Safety claims ("fireproof", "unbreakable") without testing
- Superlatives ("best", "#1") without substantiation
